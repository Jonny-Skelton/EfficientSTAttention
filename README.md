# EfficientSTAttention
Reimplementing TSMixer, RPMixer, iTransformer, and EiFormer for modeling spatial temporal data at scale.

## Goals and Setup
The goal of this project is to gain experience building models end-to-end from scratch. Each of the four models above pertain to my research interest of modeling spatial temporal data at scale.

I asked Claude to provide an architecture skeleton in comments and to include references to relevant pytorch documentation. The public source code for TSMixer, RPMixer, and iTransformer were also used as references.
